# API Routes

Express.js route definitions mapping URLs to controller logic.

-   **`auth.routes.js`**: `/api/auth` - Login and Registration.
-   **`user.routes.js`**: `/api/user` - User profile and admin user management.
-   **`fund.routes.js`**: `/api/funds` - Fund CRUD and member tracking.
-   **`payment.routes.js`**: `/api/payments` - Payment processing.
-   **`dashboard.routes.js`**: `/api/dashboard` - Stats aggregations.
