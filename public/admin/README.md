# Admin Dashboard

Contains the interface for Fund Managers.

-   **`dashboard.html`**: Overview of system stats and recent activity.
-   **`funds.html`**: Operations to Create, Edit, and Delete savings funds.
-   **`users.html`**: User management (Approvals, Removals).
-   **`tracking.html`**: The "Grid View" for tracking monthly payments per fund.
-   **`user_report.html`**: printable reports for individual members.
